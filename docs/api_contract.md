# API Contract

Phase 6 Task 2. Full FastAPI app: `backend/app/main.py`. **Interactive
OpenAPI docs are auto-generated by FastAPI** — no separate generation step:
run the server and visit `/docs` (Swagger UI) or `/redoc` (ReDoc). This
file is the human-readable contract for dashboard integration and the PPT.

All endpoints are **read-only (GET)**, served from the FROZEN Phase 3/4
models (no retraining triggered by any request) and the latest historical
snapshot per cell (no live streaming yet — Phase 7; see each endpoint's caveat).

---

## `GET /forecast`

Single-zone forecast. Provide `h3_cell` OR `lat`+`lon`.

**Query params:**
| Param | Type | Required | Description |
|---|---|---|---|
| `h3_cell` | string | one of h3_cell/lat+lon | H3 cell ID (resolution 9) |
| `lat`, `lon` | float | one of h3_cell/lat+lon | Coordinates, converted to H3 internally |
| `vehicle_type` | string | no | Override the vehicle currently being evaluated (affects recommendation escalation) |

**Response (known cell):**
```json
{
  "zone": "89618925c03ffff",
  "hotspot_probability": 0.6253,
  "predicted_count": 7.6,
  "congestion_risk": 27.92,
  "risk_band": "LOW",
  "recommendation": "Monitor",
  "confidence": 0.2506,
  "top_contributing_factors": [
    {"factor": "hotspot_probability", "contribution": 25.01},
    {"factor": "normalized_predicted_count", "contribution": 2.86}
  ],
  "is_cold_start": false,
  "last_known_event": "2024-04-04 04:36:46+00:00",
  "escalated": false
}
```

**Response (cold start — unseen cell):**
```json
{
  "zone": "ffffffffffffff",
  "hotspot_probability": null,
  "predicted_count": null,
  "congestion_risk": 0.0,
  "risk_band": "LOW",
  "recommendation": "Monitor",
  "confidence": 0.0,
  "top_contributing_factors": [],
  "is_cold_start": true,
  "note": "No historical data for this H3 cell — model is not validated to generalize confidently to unseen cells (see DECISIONS.md ADR-016). Returning a conservative default, not a fabricated prediction."
}
```

**Errors:** `400` if neither `h3_cell` nor `lat`+`lon` is provided.

**Caveat:** `hour`/`weekday`/cyclic-time features are recomputed against the
real current timestamp; density/rolling/historical-risk features are frozen
at the cell's last known historical event (no live streaming pipeline yet).

## `GET /alerts`

Live alert feed across ALL known H3 cells (2,534), not a fixed historical sample.

**Query params:**
| Param | Type | Default | Description |
|---|---|---|---|
| `level` | string | none | Filter by `GREEN`/`YELLOW`/`ORANGE`/`RED` |
| `min_band` | string | `MEDIUM` | Minimum risk band to include (`LOW`/`MEDIUM`/`HIGH`/`CRITICAL`) |
| `limit` | int | 50 | Max alerts returned (1-500), highest risk_score first |

**Response:**
```json
{
  "count": 1,
  "total_cells_evaluated": 2534,
  "alerts": [
    {
      "zone": "8960145b553ffff",
      "junction_name": "BTP040 - Elite Junction",
      "police_station": "Upparpet",
      "latitude": 12.976995,
      "longitude": 77.577297,
      "alert_level": "ORANGE",
      "probability": 0.9378,
      "risk_score": 50.34,
      "risk_band": "HIGH",
      "recommendation": "Deploy enforcement",
      "escalated": false,
      "top_contributing_factors": [
        {"factor": "hotspot_probability", "contribution": 37.51},
        {"factor": "persistence", "contribution": 8.73}
      ],
      "last_known_event": "2024-04-08 05:37:46+00:00"
    }
  ]
}
```

**Note on distribution:** this is a **per-cell snapshot** (each of 2,534
cells' single latest event), not the per-event distribution reported in
Phase 5's `docs/risk_definition.md` (44,767 validation rows, many events per
cell). The live snapshot skews much lower-risk (e.g. ~97% LOW) than the
per-event distribution (~58% LOW) — expected, not a bug: most cells' single
most-recent moment is calmer than the full event-level mix that includes
many high-activity moments.

## CORS

`allow_origins=["*"]` — intentionally permissive for this hackathon demo
(no auth, no user accounts, deployed frontend origin not fixed in advance).
Tighten to the real frontend origin before any production use beyond a demo.

- **ReDoc**: `http://localhost:8000/redoc`
- **Raw schema**: `http://localhost:8000/openapi.json`
